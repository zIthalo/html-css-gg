Tipografia

Os livros na antiguidade eram escritos por minges copistas, depois foi feito por uma prensa mecânica de movimento "criada" por Gutenberg.

A tipografia nasceu na é poca que o povo queria letras melhores de serem lidas e com bom aproveitamento de espaço(tipografia týpos[impressão] graphía [estudo{gráfio}]) a tipografia também tem a ver com emoções.

Existe a forma das emoções de fonte algumas fontes são mais indicadas para tipos de informações, como quando você quer chamar atenção para coisas importantes, para a palavra amor, para a frase feito a mão.

Estudo básico da tipografia

anatomia do tipo

O tipo é cada uma das letras cada um dos tipos de letras. Altura das minúsculas = altura x Altura das maiúsculas = altura das maiúsculas Altura das minúsculas vazadas acima= acendente Altura de todas vazadas abaixo = descendente Altura total de todas = corpo

Itens geométricos
