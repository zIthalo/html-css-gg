Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>(Não semântica)

Nesta frase tenho um termo em destaque usando a tag <strong>(Semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (Não semântica).

Nesta frase, temos um termo em ênfase usando a tag <en> (Semântica).

.

Marcando texto

Podemos criar também um texto marcado usando a tag <mark>

E no outro parágrafo, temos outro texto marcado no final

Texto grande / Texto pequeno

Estamos criando um texto grande (usando a tag <big> [não semântica])e um texto pequenousando a tag <small>[Semântica]

Texto deletado

podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag <ins>

Texto sobrescrito

Para inserir coisas do tipo x20+30 use a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O use a tag <sub>
